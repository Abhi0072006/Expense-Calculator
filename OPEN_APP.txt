⚙️  **SETUP STEPS:**
═══════════════════════════════════════════════════════════════

   Step 1: Install Node.js from nodejs.org
   
   Step 2: Open Terminal app
   
   Step 3: Go to project folder:
           cd "/Users/ayushrudra/abhirudh app"
   
   Step 4: Install dependencies:
           npm install
   
   Step 5: Start frontend (Terminal 1):
           npm run dev
   
   Step 6: Start backend (Terminal 2):
           npm run server
   
   Step 7: Open browser to:
           http://localhost:3000
